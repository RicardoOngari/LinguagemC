# 💻 Linguagem de Programação em C

Este repositório é destinado aos códigos desenvolvidos durante as aulas de **Linguagem de Programação**, utilizando a **linguagem C**.

Os programas presentes aqui foram criados com o objetivo de praticar **lógica de programação**, compreender os **fundamentos da linguagem C** e acompanhar a evolução dos conhecimentos ao longo das aulas.

## 🚀 Tecnologias Utilizadas

- Linguagem C
- Compilador GCC

## 📂 Estrutura do Repositório

Os códigos podem estar organizados por **aulas ou exercícios**.


## 📚 Conteúdos Praticados

Durante as aulas são trabalhados conceitos como:

- Variáveis e tipos de dados
- Operadores aritméticos e lógicos
- Estruturas condicionais (`if`, `else`, `switch`)
- Estruturas de repetição (`for`, `while`, `do while`)
- Funções
- Entrada e saída de dados (`printf` e `scanf`)
